## Project Description
A Java-based command-line application that manages employee data for a bank organization. The system provides functionality for sorting, searching, adding employees, and displaying organizational hierarchy using data structures and algorithms.

## Features Implemented
- File loading with employee data
- Recursive merge sort algorithm
- Binary search functionality  
- Employee management (add/display)
- Binary tree organizational hierarchy
- Menu-driven user interface

## Algorithms Used
1. **Merge Sort** - For sorting employee names alphabetically
2. **Binary Search** - For efficient employee searching
3. **Level-order Traversal** - For binary tree display

## File Requirements
- `Applicants_Form.txt` - Input file with employee names (one per line)
